# Project-A

Browser-based multiplayer strategy game where each cycle evolves via winner suggestions.

## Core Loop

- Sign in with Google
- Create one fortress
- Choose action:
  - Grow (+1 point/min)
  - Attack (-1 self / -2 target)
- Cycle lasts 72 hours
- Winner proposes one bounded change
- AI creates a PR
- Approved change deploys, then the game resets
